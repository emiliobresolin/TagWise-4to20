# Implementation Artifacts Index

This folder contains the execution-facing TagWise artifacts.

## Conventions
- Planning decisions stay in [`../planning-artifacts`](../planning-artifacts/index.md).
- Implementation-ready stories live as one file per story at the root of this folder.
- Story files use numeric BMAD-friendly keys such as `1-1-mobile-app-shell-and-sqlite-bootstrap.md` so downstream workflows can discover them cleanly.
- The aggregate planning decomposition remains in [`../planning-artifacts/story-map.md`](../planning-artifacts/story-map.md).
- A sprint tracking file is intentionally not created yet; execution tracking can start when active development begins.
